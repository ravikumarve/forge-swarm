# Forge Swarm Screenshots

This directory should contain the following screenshots for the README.md:

## Required Screenshots

1. **main-dashboard.png** - Main dashboard showing the dark theme UI with agent pipeline
2. **agent-pipeline.gif** - Animated GIF showing agents working in real-time
3. **memory-search.png** - Memory system search and browse interface
4. **code-sandbox.png** - Code sandbox execution interface
5. **templates.png** - Template quick-start buttons

## Social Preview

- **social-preview.png** - 1200x630px image for GitHub/Gumroad social preview

## How to Create Screenshots

1. Run Forge Swarm: `streamlit run forge_swarm_with_ui.py`
2. Use a screenshot tool (like Snagit, Lightshot, or built-in tools)
3. For GIFs, use a tool like LICEcap or OBS Studio
4. Ensure screenshots are high resolution (at least 1920x1080)
5. Use dark theme for consistency

## Image Specifications

- **Format**: PNG for static images, GIF for animations
- **Resolution**: Minimum 1920x1080 for screenshots
- **Social Preview**: 1200x630px (GitHub/Gumroad standard)
- **File Size**: Keep under 2MB for faster loading
